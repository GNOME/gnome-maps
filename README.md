  # GNOME Maps

  **Find places around the world**

  ![Main window map view](https://static.gnome.org/appdata/gnome-50/maps/main-light.png)

</div>

Maps is a map application for the GNOME desktop powered by [OpenStreetMap](https://www.openstreetmap.org/about).
More information is available at [apps.gnome.org/Maps](https://apps.gnome.org/Maps/).

This application is written in Javascript. It uses [GraphHopper](https://www.graphhopper.com/)'s geocoder API for search, which uses [Photon](https://photon.komoot.io/) as its backend.
GraphHopper is also used for routing (walking, cycling, and driving), while [Transitous](https://transitous.org/) is used for multimodal routing (public transit + walking).
